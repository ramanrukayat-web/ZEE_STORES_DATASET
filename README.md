## ZEE_STORES_DATASET
This file contains an analysis of Zee Stores sales data carried out using Microsoft Excel. The dataset was cleaned, analyzed, and visualized to understand sales performance, revenue patterns, and customer purchasing behavior. Excel charts and graphs were used to present insights in a clear and easy-to-understand format.

The analysis focuses on identifying trends, comparing performance across products and time periods. The visualizations help transform raw data into meaningful information that supports data-driven decision making

# Key Contents of the File


The Excel file includes:
	•	Cleaned sales dataset
	•	Calculated fields such as totals and summaries of sales
	•	Charts such as bar chart,pie chart,sparklines 

The file contains Excel charts such as:


	•	Total sales and revenue overview
	•	Sales trends over time
	•	Product or category performance comparison
	•	Summary charts highlighting key metrics


Tool Used


	•	Microsoft Excel – for data cleaning, analysis, and visualization
